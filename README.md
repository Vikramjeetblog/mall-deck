# Dubai Mall Interactive Sales Deck

A premium interactive presentation deck built using **React, Vite, Tailwind CSS, and Framer Motion**. This project was designed as a modern, non-linear, video-first commercial storytelling experience for a flagship mall destination.

It combines elegant UI, cinematic motion, expandable modules, and business-focused narratives for leasing, events, sponsorship, retail, dining, and destination opportunities.

## Project Objective

Create a presentation deck that feels more like a premium digital experience than a traditional slide deck.

The project focuses on:

* Interactive navigation
* Video-led storytelling
* Luxury visual language
* Fast and performant delivery
* Expandable architecture
* Business conversion-focused messaging

Inspired by premium brand ecosystems such as:

* Apple
* Hermès
* Tesla

* Strong typography

* High contrast layouts

## AI-Generated Assets

Generative AI was used where real assets were unavailable.

Examples include:

* Luxury mall interiors
* Sponsorship takeover visuals
* Event arena scenes
* Premium retail environments
* Pop-up activation concepts
* CTA cinematic visuals

All assets were selected, refined, and compressed for production use.



## Performance Optimizations

Implemented for speed and smoothness:

* Lazy loading of expandable modules
* Optimized MP4 video assets
* Selective video usage (not every section)
* Reduced unnecessary DOM load
* Responsive layouts
* Lightweight transitions
* Scroll-triggered motion only when needed



## Installation

Clone the repository:

```bash
git clone https://github.com/Vikramjeetblog/mall-deck
cd vite-project
```

Install dependencies:

```bash
npm install
```

Run locally:

```bash
npm run dev
```

---

## Build for Production

```bash
npm run build
```

Preview production build locally:

```bash
npm run preview
